# Python Client Consumes Public API

This example demonstrates how a client application interacts with a public API to fetch data. It simulates a program requesting specific information from a remote service without needing to understand the service's internal implementation. The script uses Python's standard library to make HTTP GET requests to a dummy API endpoint and processes the JSON response.

## Language

`python`

## How to Run

Save the code as `main.py`.
Run from your terminal: `python main.py`
